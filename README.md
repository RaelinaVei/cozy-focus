# ☕ Cozy Focus

A beautiful, minimalist Pomodoro timer designed to help you stay focused in a cozy environment. Built with Lovable and powered by modern web technologies.

## 🚀 Free Live Website!
Check out the live site here: [Cozy Focus](https://prismic-pomodoro.lovable.app/)

## ✨ Features
- **Pomodoro Timer**: Classic 25/5/15 minute intervals to maximize productivity.
- **Cozy UI/UX**: A calm, distraction-free interface designed for deep work.
- **Customizable**: Adjust timer durations to fit your personal workflow.
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile.
- **Sound Alerts**: Gentle notifications to let you know when it's time to break or get back to work.

## 🛠️ Tech Stack
This project was generated via [Lovable](https://lovable.dev) and uses:
- **React** - Frontend framework
- **Vite** - Build tool for speed
- **Tailwind CSS** - For the "cozy" styling
- **TypeScript** - For type-safe, reliable code
- **Shadcn UI** - For beautiful, accessible components
